# back_end_boilerplate

### STEPS

1. Create server.js at the root 

```bash
# Create project 
mkdir app/src
cd app 
npm init
Create src/server.js 

npm install -y

# Install express 
npm install express

# Install dotenv
sudo npm install dotenv

# Install mongoose
sudo npm install mongoose

# Install bcryptjs
sudo npm install bcryptjs

# Install jwt
sudo npm install jsonwebtoken

# Install nodemon
sudo npm install nodemon

# Install swagger 
sudo npm install swagger-ui-express

```

1. Create 
    1. middleware (auth)
    2. models(mongoose)
    3. controllers
    4. routes

1. Use post man to check the request 
2. Use mongodb cloud for the db 
3. Create .env to store the secrets..

```bash
# Run the app localy 
npm run dev
```
